## 🚀 Como Usar

### Pré-requisitos
- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Sem dependências externas ❌ Node.js
- Sem build tools necessários ❌ Webpack

### Instalação & Execução

```bash
# Clone o repositório
git clone https://github.com/Euuuller/portfolio.git
cd portfolio

# Abra no navegador (qualquer um dos métodos abaixo)
open index.html                    # macOS
start index.html                   # Windows
xdg-open index.html               # Linux

# Ou use um servidor local (Python)
python -m http.server 8000
# Acesse: http://localhost:8000
```

## 🎨 Personalização

### Adicionar Novo Projeto

Edite `js/data/data.js`:

```javascript
{
  id: 'seu-projeto',
  title: 'Título do Projeto',
  category: 'ANALYTICS',
  image: 'url-da-imagem',
  shortDesc: 'Descrição breve...',
  tags: [{ name: 'Python', class: 'project-card__tag--python' }],
  metrics: ['Métrica 1', 'Métrica 2'],
  status: 'done',
  statusLabel: 'Concluído',
  modal: {
    desafio: 'O problema...',
    solucao: 'Como resolveu...',
    impacto: 'Resultado obtido...',
    metricas: ['M1', 'M2'],
    stack: ['PYTHON', 'PANDAS'],
    urlCodigo: 'https://github.com/...',
    urlDemo: '#'
  }
}
```

### Adicionar Nova Habilidade

Edite `js/data/data.js` - array `SKILLS`:

```javascript
{
  name: 'TensorFlow',
  phrase: 'Deep Learning',
  iconUrl: 'https://cdn.jsdelivr.net/gh/devicons/devicon@latest/icons/tensorflow/tensorflow-original.svg'
}
```

### Alterar Tema de Cores

Edite `css/main.css` - variáveis CSS:

```css
:root {
  --color-primary: #your-color;
  --color-secondary: #your-color;
  /* ... */
}
```
